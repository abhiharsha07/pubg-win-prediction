# PUBG Winner Prediction using Machine Learning

This project predicts player win placement percentage using PUBG match data.

## Project Objective
To predict player win placement percentage using machine learning models based on gameplay statistics.

## Key Highlights
- Analyzed 4,000+ match records
- Performed data cleaning and feature engineering
- Built XGBoost model with R² score of 0.93
- Achieved MAE of 0.058

## Tools Used
Python, Pandas, NumPy, XGBoost

## Dataset
Dataset provided as part of Data Science training program (DataMites Institute).
